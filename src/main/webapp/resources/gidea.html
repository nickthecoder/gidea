gidea.css Style Sheet Info

table.artists

Used to create the stripy table when listing only artists

table.album, td.cover, td.tracks, td.backup

Used to layout stuff when looking at an artist or a single album

div.cover

This is bad having two styles with the same name, one used as a td, the other as a div.
This one is used when displaying an album cover - its for the set of covers at the bottom which doesn't get printed.

table.thumbnails

Used by the image and video apps to layout the thumbnail images.
One Two Three Four

a.tumbnail img

Used to ensure no padding around thumnail images (which are always links).

imageNavigation

Posistions the navigation elements in the top right of the window. Makes the links have large underlines (using a border).
Example

table.image

Positions images in the center of the screen - seems odd!

warning

Used as a span to highlight text. Currently only used to hightlight albums which have not been backed up, on the plain music catalogue. Perhaps this should be moved into global.
